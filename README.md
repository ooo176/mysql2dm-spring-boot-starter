## 2 使用说明

### 2.1 引入依赖

#### 2.1.1 添加JitPack repository

```
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

#### 2.1.2 添加依赖

```
	<dependency>
	    <groupId>com.github.ooo176</groupId>
	    <artifactId>spring-boot-starter-dm</artifactId>
	    <version>-SNAPSHOT</version>
	</dependency>
```
